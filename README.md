# chplmod


This repository contains Chapel modules written by myself. They are
quite usefl (I think) in day-to-day tasks concerning research with
datasets and scientific programming in general.

You will need to install the Chapel programming language: see https://chapel-lang.org/.


Documentation (in pdf files) is an on-going effort. I hope that what
exists will be helpful.
